## Getting started

### Common

Installing the dependencies.
```sh
npm install
```

### While development

Starting the server.

NOTE : The "dev" mode avoid caching and reload all the files on each request.

```sh
npm install
npm run start-dev
```

### While production

Starting the server.

```sh
npm install
npm run start-dev
```
